# Внести свой вклад

Любые корректные и структурированные изменения принимаются.

## Начало работы

1. Форкните репозиторий и склонируйте его:

```bash
git clone https://github.com/polytech-2024-2028/OOO-PI3DA
```

2. Добавьте upstream:

```bash
git remote add upstream https://github.com/polytech-2024-2028/OOO-PI3DA
```

## Требования к изменениям

* Работайте только в своей директории `surname/`;
* Нельзя изменять чужие папки;
* Структура должна сохраняться;

## Коммиты

* Используйте осмысленные названия (по возможности).

## Обновление ветки

```bash
git fetch upstream
git rebase upstream/main
git push --force-with-lease
```

## Pull Request

* Перед открытием убедитесь, что изменения ограничены вашей папкой;
* При нарушении структуры PR отклоняется;
* Возможен draft PR;
* В ПР кратко описать кто и что сделал.

## Важно

Если PR затрагивает файлы вне одной папки `surname/`, он считается некорректным и не принимается.
