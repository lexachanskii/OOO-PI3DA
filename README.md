Репозиторий предназначен для хранения работ студентов.

## Структура репозитория

```
.
└── sem_№/
    └── subject/
        └── surname/
            └── ...
```

* `sem_№` — номер семестра
* `subject` — название предмета
* `surname` — фамилия или ник студента
* Внутреннее содержимое не регламентируется

## Правила

* Каждый студент работает **только внутри своей папки `surname/`**;
* Допускаются изменения в разных предметах, если сохраняется одна и та же фамилия;
* Изменение файлов вне своей папки приводит к отклонению PR;
* Не добавляйте лишние файлы, не относящиеся к работе;
* Для пояснений используйте `README.md` внутри своей папки;

> **Помните, что Большой Брат следит за Вами. =)**
